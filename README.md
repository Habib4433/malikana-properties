# Malikana Properties Ltd.

বাংলাদেশের বিশ্বস্ত রিয়েল এস্টেট প্রতিষ্ঠান।

## সেটআপ

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
node server.js
```

## পেজসমূহ
- `/` - হোম
- `/plots` - জমি ও প্লট
- `/flats` - ফ্ল্যাট বিক্রয়
- `/installment` - কিস্তি ক্যালকুলেটর
- `/contact` - যোগাযোগ
- `/login` - লগইন/রেজিস্ট্রেশন
- `/dashboard` - ক্রেতার ড্যাশবোর্ড
- `/admin` - অ্যাডমিন প্যানেল
